# Flux System

This directory contains the Flux CD bootstrap manifests.

## Initial Setup

`gotk-components.yaml` is generated by the Flux CLI and should NOT be hand-edited.
Generate or update it with:

```bash
flux install --export > k8s/flux-system/gotk-components.yaml
```

Then bootstrap onto the cluster:

```bash
# Create the deploy key secret (Flux will manage the GitHub deploy key)
flux bootstrap github \
  --owner=aidenscott2016 \
  --repository=nixos \
  --branch=main \
  --path=k8s/flux-system \
  --personal
```

## SOPS Age Secret

After bootstrap, create the age key secret for SOPS decryption.
The age private key is the same one used by agenix (from /etc/ssh/ssh_host_ed25519_key):

```bash
# On bes, derive age key from host SSH key (if using ssh-to-age)
ssh-to-age -private-key < /etc/ssh/ssh_host_ed25519_key > /tmp/age.key

kubectl create secret generic sops-age \
  --namespace=flux-system \
  --from-file=age.agekey=/tmp/age.key

rm /tmp/age.key
```
